# 🌍 TripPulse ✨

## AI-Powered Travel Planner

TripPulse is a full-stack travel planning application that helps users generate personalized trip itineraries, view destination maps, check weather information, save trips, and download trip plans as PDF.

---

## 🚀 Features

- 🧠 AI-ready trip itinerary generation
- 📅 Day-wise travel planning
- 📍 Google Maps destination view
- 💰 Budget breakdown
- ❤️ Save trips using MongoDB
- 📄 Download trip plan as PDF
- 🌦️ Weather integration
- 💬 AI chat assistant-ready structure
- 📱 Fully responsive UI
- ✨ Modern animated interface
